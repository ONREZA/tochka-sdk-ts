# Клиенты

Для работы с клиентами должно быть выдано разрешение `ReadCustomerData`.

С помощью метода [Get Customers List](https://developers.tochka.com/docs/tochka-api/api/get-customers-list-open-banking-v-1-0-customers-get) вы получите список подключённых компаний, а с [Get Customer Info](https://developers.tochka.com/docs/tochka-api/api/get-customer-info-open-banking-v-1-0-customers-customer-code-get) — детальную информацию по каждому клиенту.
