# Счета

Для работы со счетами должны быть выданы разрешения `ReadAccountsBasic`, `ReadAccountsDetail`.

-   [Get Accounts List](https://developers.tochka.com/docs/tochka-api/api/get-accounts-list-open-banking-v-1-0-accounts-get) — получить информацию по всем счетам
-   [Get Account Info](https://developers.tochka.com/docs/tochka-api/api/get-account-info-open-banking-v-1-0-accounts-account-id-get) — получить информацию по одному счёту
