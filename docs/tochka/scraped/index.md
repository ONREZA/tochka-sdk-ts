# Приём платежей

API приёма платежей позволяет принимать платежи банковскими картами и через СБП напрямую, без использования стандартной платёжной формы банка.

Чтобы использовать API приёма платежей, вам нужны:

-   Собственный пользовательский интерфейс для ввода платёжных данных
-   Действующий сертификат AOC, подтверждающий соответствие стандарту [PCI DSS](https://www.pcisecuritystandards.org/minisite/ru-ru/)

> **ИНФО**
> 
> Если у вас нет своего интерфейсного решения или сертификата, обратитесь к [API работы с платёжными ссылками](https://developers.tochka.com/docs/tochka-api/api/rabota-s-platyozhnymi-ssylkami).
