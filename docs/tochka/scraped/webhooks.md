# Вебхуки

> **ИНФО**
> 
> Пока что вебхуки работают только с операциями по СБП.

Вебхук — это POST-запрос с информацией о событии. Тело запроса содержит строку — JWT-токен, подписанный алгоритмом RS256. Вебхук отправляется с заголовком `Content-Type: text/plain`.

Чтобы убедиться, что вебхук пришел от Точка Банка, используйте наш [публичный ключ OpenAPI](https://enter.tochka.com/doc/openapi/static/keys/public).

## Получение вебхуков

### При онбординге

Передайте персональному менеджеру URL, на который вы хотите получать вебхуки. Все события будут отправляться на этот URL.

### При выполнении операций

При выполнении операции можно передать параметр `callbackUrl`. Все связанные с операцией вебхуки будут доставляться на этот URL.

> **ИНФО**
> 
> Если оба способа настроены одновременно, приоритет имеет URL, указанный при выполнении операции (`callbackUrl`).
